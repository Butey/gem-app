веб-интерфейс в стиле dark glassmorphism для энциклопедии минералов.

Цветовая схема:
- Фон: радиальный градиент сланца (#1a1c1e → #0d0e10)
- Акцент 1: изумрудный/малахитовый (#10b981)
- Акцент 2: фиолетовый/аметистовый (#a855f7)
- Текст: светлый серо-голубой (#e2e8f0)

Glassmorphism эффекты:
- Карточки: rgba(255,255,255,0.03) + backdrop-filter: blur(12px)
- Границы: 1px solid rgba(255,255,255,0.1)
- Hover: граница меняет цвет на акцентный + свечение box-shadow

Компоненты:
- Header: заголовок + поисковая строка (rounded-full, полупрозрачный фон)
- Категории: карточки с цветными заголовками
- Алфавитная навигация: горизонтальный ряд букв
- Grid карточек: 4 колонки (desktop), 2 (tablet), 1 (mobile)
- Карточка минерала: изображение + категория + название

Анимации:
- transition: all 0.3s ease
- Изображения: grayscale-[0.5] → grayscale-0 + scale при hover
- Плавные переходы цветов и теней

Адаптивность:
- Mobile-first подход
- Tailwind CSS классы для брейкпоинтов
- Сохранение стиля на всех устройствах

Типографика:
- Шрифт: Inter (Google Fonts)
- Заголовки: font-light, tracking-wider
- Мелкий текст: uppercase, tracking-[0.2em]
