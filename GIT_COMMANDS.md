# Git - Быстрые команды для управления версиями

## Основные команды

### Посмотреть историю коммитов
```bash
cd /opt/gems_app && git log --oneline -10
```
Показывает последние 10 коммитов с хешами.

---

### Посмотреть текущие изменения
```bash
git status              # список изменённых файлов
git diff                # содержимое изменений (незакоммиченные)
git diff --staged       # содержимое изменений (подготовленные к коммиту)
```

---

### Вернуться к предыдущей версии

**Отменить все незакоммиченные изменения:**
```bash
git checkout HEAD -- .
```

**Вернуться к конкретному коммиту:**
```bash
git checkout <hash> -- .
# Пример: git checkout a1b2c3d -- .
```

**Полный откат к предыдущему коммиту:**
```bash
git reset --hard HEAD~1
```

---

### Создать точку возврата (бранч)
```bash
git branch backup-before-changes    # создать ветку
git checkout backup-before-changes  # переключиться
```

**Вернуться на main:**
```bash
git checkout main
```

---

### Список всех веток
```bash
git branch -a
```

---

## Готовые фразы для Qwen Code

Просто скажи:
- **"Верни прошлую версию"** → откатит к последнему коммиту
- **"Что изменилось?"** → покажет текущие изменения
- **"Покажи историю"** → покажет последние коммиты
- **"Создай точку возврата"** → создаст бранч
