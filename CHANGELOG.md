# Changelog

Все изменения в проекте "Gems Encyclopedia App".

## [1.0.3] - 2026-04-15

### Админ-панель
- **Страница входа**: отдельный шаблон без base.html, минималистичный дизайн
- **Админ-дашборд**: переключатель тем, кнопка "Пользователи"
- **Управление пользователями**: создание/удаление, хэширование паролей
- **Модель User**: таблица users, werkzeug.security для паролей
- **Скрипт**: `create_admin_user.py` для создания пользователей
- **Fix**: logout перенаправляет на welcome

### Контрастность
- **Страница минерала**: все цвета заменены на CSS-переменные
- **Текст**: var(--text-primary), var(--text-secondary), var(--text-muted)
- **Карточки**: var(--card-bg), var(--card-border)

## [1.0.0-beta] - 2026-03-22

### Добавлено
- **Стартовая страница** — приветственный экран с категориями
- **Каталог минералов** — `/catalog` с фильтрами и алфавитным указателем
- **Алфавитный указатель** — навигация по первой букве
- **Меню категорий** — 7 категорий второго уровня
- **Шкала Мооса** — визуальная шкала твёрдости с цветовым градиентом
- **Галерея фото** — поддержка нескольких изображений для минерала
- **Лайтбокс** — просмотр фото в полном размере
- **Цветовые свотчи** — отображение цветов минералов
- **Поиск с подсказками** — FTS5 + префиксный поиск
- **Админ-панель** — CRUD для минералов

### Изменения
- Категории в множественном числе (поделочные, органические, полудрагоценные)
- Увеличено главное фото до 500px
- Двухколоночная сетка характеристик

### Исправлено
- Поиск для коротких кириллических запросов
- Отображение категорий в хлебных крошках

### Технические
- Python 3.9+, Flask 2.x, SQLite + FTS5
- systemd служба `gems_app.service`
- Gunicorn с 4 workers
